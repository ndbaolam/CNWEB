Thông tin môn học

IT4409 Công nghệ web và dịch vụ trực tuyến (Web technologies and e-Services)

  • - Khối lượng (Credits): 3(2-2-0-6)
  • - Học phần tiên quyết (Prerequisite): Không (None)
  • - Học phần học trước (Pre-courses): (IT3090 hoặc/or IT3091) và/and (IT3100 hoặc/or IT3600)
  • - Học phần song hành (Corequisite Courses): Không (None)

Mục tiêu

Học phần cung cấp các kiến thức cần thiết để xây dựng và phát triển các ứng dụng, dịch vụ trực tuyến trên môi trường web. Sau khi học xong học phần này, sinh viên có khả năng:
  • - Có thể sử dụng một ngôn ngữ lập trình như JSP, PHP để phát triển ứng dụng web cho các lĩnh vực khác nhau như: hệ quản trị nội dung (CMS), cổng thông tin (Portal), hay các ứng dụng điện tử (eCommerce, eLearning, eGoverment,…)
  • - Nắm vững được các công nghệ web, các platform, framework hiện đang thịnh hành.
Ngoài ra học phần cũng cung cấp cho sinh viên các kỹ năng làm việc nhóm, thuyết trình và thái độ cần thiết để làm việc và ứng dụng các kỹ thuật trong các dự án, công ty về phần mềm sau này.

Course Information

The course provides students with the necessary knowledge to build and develop online applications and services on the web environment. After completing this course, students are able to:
  • - Use a programming language such as JSP, PHP to develop web applications for different areas such as content management system (CMS), portal, or online applications (eCommerce, eLearning, eGoverment, ...)
  • - Understand the popular web technologies, platforms, frameworks.
In addition, the course also provides students with skills of teamwork and presentation, and attitudes needed to work and apply the techniques in software projects and companies in the future.

Các Công Nghệ Web

Công Nghệ Mô Tả
Node.js Node.js là một nền tảng phát triển ứng dụng phía server, dựa trên JavaScript, có thể sử dụng để xây dựng các ứng dụng web nhanh chóng và có khả năng mở rộng.
React.js React.js là một thư viện JavaScript mạnh mẽ, do Facebook phát triển, dùng để xây dựng giao diện người dùng với tính năng quản lý trạng thái linh hoạt và khả năng phát triển ứng dụng nhanh chóng.
Vue.js Vue.js là một framework JavaScript tiến bộ, dùng để xây dựng giao diện người dùng. Nó dễ học và có cấu trúc linh hoạt, giúp phát triển ứng dụng front-end hiệu quả.
Next.js Next.js là một framework React mạnh mẽ, cho phép phát triển các ứng dụng React với khả năng render phía server (SSR) và tính năng tối ưu hóa SEO tốt.
Cloudinary Cloudinary là dịch vụ lưu trữ và quản lý hình ảnh, video trên đám mây. Nó cung cấp các công cụ tối ưu hóa hình ảnh và video, giúp giảm tải cho hệ thống và cải thiện trải nghiệm người dùng.
Apache Apache là một trong những máy chủ web phổ biến nhất trên thế giới, được sử dụng để phục vụ các trang web tĩnh và động với cấu hình linh hoạt và dễ sử dụng.
Docker Docker là một nền tảng mã nguồn mở, cho phép phát triển, vận hành và quản lý các ứng dụng dưới dạng container. Nó giúp việc triển khai ứng dụng dễ dàng và nhanh chóng.
Git Git là một hệ thống quản lý phiên bản phân tán, giúp các nhà phát triển theo dõi và quản lý lịch sử mã nguồn, hỗ trợ làm việc nhóm hiệu quả.
Github Github là một nền tảng lưu trữ mã nguồn sử dụng Git. Nó cung cấp các công cụ hỗ trợ phát triển phần mềm, quản lý dự án và cộng tác trực tuyến.
